# 🎫 AI-Assisted Ticket Triage Automation
This project simulates an automation workflow used in IT and Identity & Access Management (IAM) environments. It reads incoming support tickets, classifies them by category, assigns priority levels, and generates AI-style summaries.

## Features
- Reads ticket data from JSON
- Classifies tickets into categories (Access, Password, System Issue, Other)
- Assigns priority levels based on keywords
- Generates AI-style summaries for each ticket
- Beginner-friendly Python automation

## Technologies Used
- Python
- JSON
- Basic automation logic
- AI-style summarization (placeholder function)
